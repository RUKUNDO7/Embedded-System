# Rukundo Project Structure

This project now runs from the repository root (`R:\Embedded`).

```text
R:\Embedded

  rukundo.db

```

## Run the Dashboard

From `R:\Embedded`:

```bash
python app.py
```

Then open:

- `http://127.0.0.1:9215/`
